# Constructor

`Constructor` contains the external construction system: Steward, Settlers, Territories, Settlements, Forge MCP, prompt assets, derived memory, and TypeScript construction tooling.

This is not Tier 5. It is outside the HealthOS clinical/runtime hierarchy and must not be described as Core, GOS, Runtime, Boundary, Stage, clinical authority, merge authority, or production automation.

Current contents:

- `Steward/` - derived Steward state, memory, prompts, sessions, handoffs, and policies.
- `Settler/` - construction metadata for territories, settlers, and settlements.
- `Territory/` - repository-local territory records.
- `ts/` - TypeScript packages for Steward, Forge MCP, contracts, and managed-agent seams.

Official docs remain canonical. Constructor memory is a derived index only.
