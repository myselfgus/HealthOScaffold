# Skill: Scaffolding

## Overview
Use this skill to bring macOS SwiftUI apps into the modern macOS design system.

## Guidelines
- Start with standard structures (NavigationSplitView, TabView, etc.).
- Avoid bespoke chrome unless needed.
- Use system-adaptive colors and materials.
- Split files by responsibility (App, Views, Models, Stores, Services).
- Avoid monolithic View files.
