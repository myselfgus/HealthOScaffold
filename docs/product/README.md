# Product specifications

This directory contains consolidated technical product specifications for HealthOS.

Architecture documents under `docs/architecture/` remain canonical doctrine.

Execution documents under `docs/execution/` remain canonical governance, sequencing, status tracking, and implementation discipline.

Product specifications in this directory consolidate the current technical product definition for builders across runtimes, interfaces, contracts, and implementation maturity.

Product specifications must not overclaim implementation maturity.

Each product specification must cite and link back to the relevant architecture and execution documents.
