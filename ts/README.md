# TypeScript workspace

Contains async services, APIs, MCP adapters, and service orchestration.

HealthOScaffold is the historical repository name and initial scaffolding phase for HealthOS. All implemented TypeScript contracts, runtimes, apps, tests, and documentation in this repository are part of HealthOS unless explicitly marked experimental or deprecated. "Scaffold" describes maturity, not project identity.

Workspace packages include `@healthos/steward` for repository engineering stewardship CLI scaffolding.
